## About the API

This API stores and retrieves an analysis of cross references from an SAP backend 

## Available routes
```
POST    api/login
POST    api/logout
POST    api/register
GET     api/xrefs
POST    api/xrefs
GET     api/xrefs/search/{name}
GET     api/xrefs/{id}
PUT     api/xrefs/{id}
DELETE  api/xrefs/{id}
```
